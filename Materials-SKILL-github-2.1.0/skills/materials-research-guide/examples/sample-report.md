# 缩略演示：TiO2 光催化研究设计

> 本文件只演示输出结构和证据标记，不是可直接执行的完整实验方案。任何具体合成与测试条件都应在重新检索直接文献、核对设备 SOP 和完成 EHS 审核后确定。

## 研究问题与边界

核心问题：在相同晶相、比表面积和反应器边界下，氮改性是否改变 TiO2 的光响应或载流子行为，并因此改善目标反应？经典工作建立了 TiO2 半导体光化学的研究基础 `[E1]`，综述总结了半导体光催化的环境应用和主要反应路径 `[E2]`，TiO2 纳米材料的制备、性质与改性已有系统综述 `[E3]`。这些来源不足以单独证明本示例中“氮改性必然有效”。

## 证据缺口与置信度

- 高置信度：TiO2 是成熟的半导体光化学材料体系 `[E1–E3]`。
- 中置信度：晶相、缺陷、表面化学和载流子行为可能影响光催化响应 `[E2, E3]`。
- 低置信度：当前证据不足以判断选定氮改性路线、掺杂状态或性能增益。
- 必补：氮改性直接原始研究、目标反应的质量平衡方法、光子通量归一化方法和失活证据。

## 实验方案

1. 文献筛选阶段：确定可复现的未改性基线与至少两种有直接来源的改性窗口。
2. 样品阶段：同批制备基线与改性样；设置无催化剂、暗态、未改性和吸附平衡对照。
3. 结构筛选阶段：先确认晶相、比表面积、吸收边和表面化学，避免把结构差异误判为“改性效应”。
4. 性能阶段：在相同光场、催化剂质量、初始浓度、传质和取样规则下比较；优先报告动力学适配范围、碳/元素质量平衡和重复性。
5. 机制阶段：仅在性能差异可重复后开展捕获、原位或时间分辨实验。

具体温度、前驱体比例、处理时间和光强均为“待直接文献核验”，本示例不提供无依据单点值。

## 表征方案

| 科学问题 | 方法 | 样品/条件 | 可观察量 | 支持/否定的假设 | 局限与互证 |
|---|---|---|---|---|---|
| 晶相是否一致 | XRD | 干燥粉体 | 相组成、晶粒相关峰宽 | 排除相变主导差异 | 对低含量/无定形相不敏感；配合 Raman/TEM |
| 氮及缺陷状态是什么 | XPS | 表面、充电校正 | N 1s、Ti 2p、O 1s | 支持表面化学变化 | 表面敏感且峰拟合非唯一；需参考样和互证 |
| 光吸收是否变化 | UV-Vis DRS | 相同装样条件 | 反射谱、吸收边 | 支持光学响应变化 | 吸收增强不等于有效载流子增加 |
| 复合行为是否变化 | 稳态/时间分辨 PL | 相同激发和光学密度 | 发光强度、寿命 | 提供载流子过程线索 | PL 不能单独证明分离效率；需电化学或瞬态互证 |

## 机理假设与证伪

- H1：改性改变电子结构或缺陷态，从而影响可利用光响应。证伪：吸收变化存在但量子效率、载流子动力学和反应速率均无一致改善。
- H2：性能差异主要来自比表面积、晶相或吸附，而非电子结构。证伪：在这些变量被匹配后差异仍稳定存在，并获得独立电子结构证据。

当前只能把 H1/H2 作为竞争假设，不能写成已证明机理。

## 创新点评估

候选创新点应放在“直接文献检索完成后”比较：改性化学是否新、是否提供更强因果证据、是否在更严格的光场与质量平衡下验证。现阶段不主张“首次氮改性 TiO2”。
