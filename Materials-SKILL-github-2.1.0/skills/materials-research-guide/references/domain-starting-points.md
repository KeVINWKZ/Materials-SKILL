# 常见领域工程起点与标准入口

本表只在**没有直接文献参数**时使用。所有起点默认标 `[S]`；相邻体系迁移标 `[X]`。标准用于统一试样、测试和报告方式，不证明某个配方或条件适合当前材料。标准会修订，引用前须在官方页面核对最新版本、适用范围和实验室许可。

## 通用起点

- 样品：空白/未处理基线 + 低/中/高三个筛选水平；中心点来自安全边界内的 SOP、设备推荐或最接近体系。
- 工艺：先筛单因素方向，再在有效窗口内研究交互作用；不给无来源的单点“最佳值”。
- 测试：固定试样制备、几何、环境、校准和数据处理；标准方法与实际服役条件分别报告。

## 领域导航

| 领域 | 第一轮筛选维度与最小对照 | 官方方法入口（先核对适用范围） |
|---|---|---|
| 粉体/多孔材料/催化剂 | 未处理或载体空白；分散/热处理低中高；粒径、比表面积、相组成和活性同时跟踪 | [ISO 13320 粒度激光衍射](https://www.iso.org/standard/69111.html)；[ISO 9277 BET 比表面积](https://www.iso.org/standard/71014.html) |
| 摩擦学/润滑/耐磨涂层 | 基材与空白润滑剂；载荷、速度、距离/时间低中高；固定接触几何、温湿度和供油方式 | [ASTM G99 旋转滑动磨损](https://store.astm.org/standards/g99)；[ASTM G133 往复滑动磨损](https://store.astm.org/standards/g133) |
| 腐蚀/防护涂层 | 裸基材、涂层空白和介质空白；暴露时间/温度/介质强度分级；同步记录表面状态和局部腐蚀 | [ASTM G31 浸泡腐蚀](https://store.astm.org/standards/g31)；[ASTM G59 极化电阻](https://store.astm.org/standards/g59)；[ASTM D3359 涂层胶带附着](https://store.astm.org/standards/d3359)；[ISO 2409 划格](https://www.iso.org/standard/76041.html) |
| 聚合物 | 未填充基体；添加量和加工条件低中高；固定调湿、试样制备、温度和应变速率 | [ASTM D638 塑料拉伸](https://store.astm.org/standards/d638)；[ISO 527-1 塑料拉伸总则](https://www.iso.org/standard/75824.html) |
| 金属/合金 | 原始态或标准热处理基线；工艺温度/时间/变形量分级；固定取样方向与表面制备 | [ASTM E8/E8M 金属拉伸](https://store.astm.org/standards/e8)；[ISO 6507-1 维氏硬度](https://www.iso.org/standard/83898.html) |
| 先进陶瓷 | 原始/未烧结或基准烧结样；成形、烧结和气氛分级；密度、缺陷、晶粒与力学响应联测 | [ASTM C1161 室温弯曲强度](https://store.astm.org/standards/c1161) |
| 聚合物基复合材料 | 纯基体/未增强基线；增强体含量、铺层或界面处理分级；固定方向、厚度和调湿 | [ASTM D3039/D3039M 复合材料拉伸](https://store.astm.org/standards/d3039) |
| 二次电池/电化学储能 | 空白电极或基线配方；先形成/稳定再做倍率与循环；必须报告质量负载、面积容量、电压窗、温度和电解液条件 | [IEC 62660-1 车用锂离子电芯性能](https://webstore.iec.ch/en/publication/28965)；[IEC 62620 工业二次锂电池](https://webstore.iec.ch/en/publication/7268)。仅在应用范围匹配时使用 |

## 使用限制

若官方标准与材料形态、设备、尺度或服役环境不匹配，只把它作为报告字段和方法设计线索。高温、高压、强腐蚀、易燃电解液、反应性粉体或纳米暴露场景必须服从 EHS、SDS 和设备 SOP；本表不提供绕过安全评估的操作参数。
