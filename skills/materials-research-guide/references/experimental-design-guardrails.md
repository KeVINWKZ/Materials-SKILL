# 实验设计护栏

## 假设到实验的映射

对每条假设回答：

1. 改变哪个独立变量？
2. 保持哪些混杂因素不变？
3. 需要哪些空白、阴性、阳性和基线对照？
4. 观察什么响应量？
5. 什么结果支持、削弱或否定假设？
6. 结果不确定时，下一项判别实验是什么？

## 参数规则

- 有直接文献窗口时，给出来源和适用边界。
- 仅有相邻体系时，给出保守筛选范围，不给单点“最佳值”。
- 没有文献依据时，标注“工程起始值”，解释选择逻辑并安排小规模预实验。
- 不把优化结果提前写成预期事实。

### 无直接文献时的工程起点规则

只有在检索过直接体系仍无结果，或当前无法实时检索且已明确降级时，才使用以下规则：

1. 先锁定硬边界：EHS、SDS、设备量程、材料稳定区、机构 SOP 和样品量；安全边界优先于所有类比。
2. 选择最接近的相邻体系或官方方法标准，列出材料形态、尺度、界面、环境和设备差异；可迁移部分标 `[X]`，其余不得照搬。
3. 用“基线/低/中/高”窗口代替一个看似精确的最佳值。中心点来自 SOP、设备推荐、供应商方法或相邻文献，全部标 `[S]`；若这些来源也没有，只给选择原则，不编数字。
4. 第一阶段只验证可行性与方向，包含空白、未处理/单组分基线和至少一个失效判据；确认信号高于噪声后再做交互作用或响应面优化。
5. 每轮只扩大一个有依据的边界，并设置停止条件。跨材料形态、尺度、反应器、载荷或环境迁移时，重新做小规模筛选。
6. 报告工程起点的依据、适用边界和升级条件；实验结果出现后，用真实数据替换 `[S]`，不得追溯性伪装成 `[D]`。

常见领域的筛选维度和标准方法入口见 [domain-starting-points.md](domain-starting-points.md)。标准用于统一测试方法，不自动证明某个参数适合当前体系。

## 对照与重复

根据问题选择：未处理基线、单组分、载体/基体、暗态、无催化剂、无载荷、空白溶剂、已知阳性标准或工艺空白。区分技术重复和独立制备重复。若用户未给统计计划，把“三个独立批次”作为常见起点而非普适硬规则，并根据成本、变异和标准要求调整。

## 表征互证

至少使用两类互补证据支持关键机理。例如：

- 晶相：衍射 + 光谱/显微；
- 化学态：XPS/XAS + 参考样或深度信息；
- 界面：高分辨成像 + 元素分布 + 能带/电荷证据；
- 反应物种：捕获实验 + 原位/操作谱学；
- 摩擦磨损：摩擦曲线 + 三维形貌 + 磨屑/转移膜化学；
- 力学强化：显微结构 + 缺陷/相分析 + 断口证据。

每种方法都要写明其不能单独证明什么，以及常见伪影。

## 创新性

把候选创新点与最接近的 3–5 项工作按材料、工艺、机理证据、性能、边界条件和可重复性逐项比较。创新点应写成可验证差异，不写营销口号。没有系统检索时，将创新性结论降级为“候选创新点”。

## 安全与伦理

- 识别毒性、腐蚀、易燃、爆炸、加压、高温、纳米粉尘、强光/激光、电气和废物风险。
- 对高风险材料或步骤，要求用户遵循机构 EHS、SDS、设备 SOP 和专业监督。
- 不凭空给出高风险操作的可执行参数。
- 标明环境释放、纳米材料暴露、数据完整性和图像处理风险。

## 可复现性

记录批次、纯度、供应商信息、前处理、环境、设备型号、校准、原始数据路径、排除标准和分析代码版本。优先设计能区分材料效应与测量伪影的验证实验。
